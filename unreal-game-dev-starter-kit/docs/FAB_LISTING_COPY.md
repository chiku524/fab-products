## Long description

**Unreal Game Dev Starter Kit** is a **documentation-first** pack for developers, designers, and small studios who want a **shared baseline** before content and code sprawl make everything expensive to fix.

**What you get**

- **Architecture primer** — Modules, Game Instance / Mode / State, subsystems, and sensible Blueprint vs. C++ split.  
- **Workflow guide** — From prototype to vertical slice with clear exit criteria.  
- **Source control** — `.gitignore` patterns, Git LFS sample, branching habits for Unreal’s binary-heavy workflow.  
- **Templates** — Merge-ready ignore fragments, LFS attributes sample, new-project checklist.  
- **Fab-ready files** — License, changelog, and support placeholders you can adapt.

**What it is not**

- Not an Unreal plugin or marketplace content pack.  
- Not a substitute for Epic’s official documentation or hands-on courses.  
- Not legal or production-management consulting.

**Engine focus:** Unreal Engine **5.x** (principles often apply to UE4 with judgment).
