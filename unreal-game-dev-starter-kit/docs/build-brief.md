# Build brief — Unreal Game Dev Starter Kit

**Product type:** Documentation + templates (no engine binary)  
**Goal:** Give Fab buyers a **credible, reusable baseline** for UE5 game project organization and team workflow.

## v1 scope

| Included | Rationale |
|----------|-----------|
| Architecture primer | Most common vocabulary gaps on small teams |
| Milestone workflow | Prevents eternal “prototype” with no exit |
| Source control chapter | Biggest preventable pain in Unreal + Git |
| Copy-paste templates | Immediate utility after purchase |
| Fab listing/support skeleton | Faster publisher time-to-listing |

## Non-goals (v1)

- Engine plugins or sample `.uproject` in the zip  
- Genre-specific combat / AI / networking tutorials  
- Legal contracts or hiring templates  

## Success criteria

- A new lead programmer reads **one chapter** and says “we should adopt this naming convention.”  
- A producer can **export** the docs to PDF for an investor update without embarrassment.
