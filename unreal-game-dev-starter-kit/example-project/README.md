# Example project (optional)

This starter kit **does not ship** a compiled Unreal project—buyers apply the guides to **their** game.

## If you (the publisher) want a demo

1. Create a **blank** or **third-person C++** template project in UE5 for marketing screenshots or trailer capture.  
2. Apply the kit’s `Content/` naming suggestions so gallery images look coherent.  
3. Host that project **separately** (zip, cloud link) if the Fab listing offers an “example” download—do not duplicate the entire documentation pack inside the project zip unless you intend to.

## Relationship to this repository

Keep example projects **out of** the main documentation zip if you want a small buyer download; link the example from the Fab description instead.
