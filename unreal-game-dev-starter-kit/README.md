# Unreal Game Dev Starter Kit

A **Fab-ready documentation and template pack** for teams and solo developers starting (or leveling up) a game project in **Unreal Engine**. It is **not** an engine plugin: you get **guides**, **checklists**, and **copy-paste templates** (`.gitignore` fragments, Git LFS hints, project conventions) you can apply to any UE **5.x** project—C++ or Blueprint-first.

**Positioning:** “Ship faster with fewer avoidable mistakes—structure, workflow, and discipline before you drown in assets.”

This folder is one **product** inside the [fab-products monorepo](../README.md).

## Audience

- New UE teams who want a **shared baseline** for folders, naming, and reviews.  
- Experienced developers who want a **compact onboarding pack** for contractors.  
- Indies who need a **reality-checked pipeline** from prototype to first milestone.

## Engine scope

Written for **Unreal Engine 5.4 and newer** (5.x). Older engines may share workflow ideas; this kit does not target UE4 in detail.
