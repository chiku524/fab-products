## Long description

**Immersive Labs Unreal Engine Development Bundle** is a **multi-file** package for teams and indies standardizing on **Unreal Engine 5.4 and newer**.

**Included**

- **Level Selection Sets** — Save and recall **named actor selections** in the level editor.  
- **WorldBuilder Audit & Convert** — Audit-oriented workflow for world-building cleanup and instancing-oriented conversion (per shipped feature set).  
- **Unreal Game Dev Starter Kit** — Markdown guides and templates for **project structure**, **milestones**, and **Git/LFS** habits (no plugin).

**Important**

- **Multiple downloadables** — You will download **separate files** for each plugin (and the starter kit). Use the plugin build that matches **your exact UE minor line** (e.g. 5.4.x with 5.4.x).  
- **Windows 64-bit editor** for the code plugins unless your listing states otherwise.  
- **Minimum engine:** **UE 5.4**.
