# Harbor Suite — support

**Publisher:** Immersive Labs  
**Contact (Fab listing):** `mailto:nico.builds@outlook.com`

## Scope

Reasonable-effort help with:

- Installing the plugin into a C++ Unreal project.
- First successful compile and editor load.
- Harbor Suite panel behavior as documented in `docs/TESTING.md`.

## Out of scope (unless separately agreed)

- Custom feature development for a specific title.
- Debugging unrelated project code, third-party plugins, or engine forks.
- Legal / trademark advice (see `docs/TRADEMARK_NOTE.md`).

## Reporting issues

Include:

- Unreal **exact version** (for example 5.7.1).
- **Windows** build and Visual Studio version.
- Steps to reproduce, **Output Log** errors mentioning `HarborSuite`, and whether you tested the **packaged** plugin or only the git checkout.
