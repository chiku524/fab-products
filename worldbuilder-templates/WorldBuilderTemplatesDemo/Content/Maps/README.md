# Maps (game project)

## This folder

The **World Builder Templates** product ships its starter **`.umap`** files **in the plugin**, not in this project’s `Content/Maps/`. Browse to:

- **`/WorldBuilderTemplates/...`** in the **Content Browser** (plugin), or on disk:  
  **`Plugins/WorldBuilderTemplates/Content/Maps/`**

See **`../../../docs/SAMPLE_MAPS.md`** (from the **worldbuilder-templates** product in this monorepo) for how to use the template maps.

## Your content

Add any **local** test maps for this project under this folder. Unreal **`.umap`** binaries are not required to be empty—keep your team’s policy on committing maps.
