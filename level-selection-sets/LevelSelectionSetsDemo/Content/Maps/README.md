# Maps

## Why this folder is empty

Unreal **`.umap`** files are binary. This repository keeps the **sample game project** lightweight. Create your own test maps here, or use **Engine** default maps (see `Config/DefaultEngine.ini`).

## Try the plugin

1. Open **`LevelSelectionSetsDemo.uproject`**.
2. Create a level, place actors, and use the **Level Selection Sets** tools from the product UI (see `Plugins/LevelSelectionSets/docs/TESTING.md`).

## Default maps (optional)

You can set **Editor Startup Map** in **Project Settings → Maps & Modes** to a level you save under `/Game/Maps/`.
