## Long description (listing body)

WorldBuilder Audit & Convert is an Unreal Engine **editor plugin** for environment artists, level designers, and indie teams who want **cleaner, faster levels** without spending hours on manual scene cleanup.

**What you get**

- **Scene audit** — Scan the current level or a selection for repeated meshes, grouping opportunities, and practical cleanup hints.  
- **Smart conversion** — Turn eligible repeated static mesh actors toward **Instanced Static Mesh** / **Hierarchical ISM** structures where it makes sense.  
- **Presets** — Forest, modular city, interior props, and mobile/VR-friendly rule profiles so non-specialists can get results quickly.  
- **Safety first** — **Dry-run preview** before you commit changes; session revert where supported.  
- **Reports** — Export **CSV** or **JSON** for clients, producers, or internal review.

**Who it’s for**

Builders who want **optimization for artists**, not only for engineers: polish passes, performance sweeps, outsource cleanup, and late-stage shipping.

**Important**

- Editor-only workflow; not a replacement for Unreal’s full profiler.  
- Use the plugin package that matches your **exact** Unreal Engine **5.4+** editor version.  
- **Windows 64-bit editor** supported for v1 unless your listing states broader support.
