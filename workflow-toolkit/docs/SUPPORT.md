| Packaged plugin from `RunUAT BuildPlugin` (not only raw monorepo source) | Use for shipping QA |

## Before you write

1. **Engine version** (exact patch, e.g. 5.7.1).  
2. **Plugin / VersionName** in `WorkflowToolkit.uplugin`.  
3. **Steps to reproduce** and **expected vs actual** behavior.  
4. **Output Log** (red errors) from **Window → Developer Tools → Output Log**.  
5. Whether the issue is in **editor** (panel) vs **PIE** (runtime subsystem).
