# Maps

## Why this folder is empty

This sample project does not ship a demo `.umap` in the repository. You can add maps under this folder, or use **DefaultEngine** **Entry** maps for quick **PIE** tests of **Workflow Toolkit** (editor and runtime subsystems).

## Quick test

1. Open **`WorkflowToolkitDemo.uproject`**.
2. **Play** in Editor to exercise **runtime** features; use the **Workflow Toolkit** panel for editor workflows.
